# Cryptoglyph-Necrotek

Cryptoglyph-Necrotek is a Necron-inspired, browser-based glyph renderer for the “Cryptic Language” script style. It supports PNG export and clean SVG export (no glow/blur filters). Runs locally via `file://` and can be hosted on GitHub Pages.

## Features
- Real-time glyph rendering (Canvas)
- Resizable preview pane
- Optional column wrap layout
- Export: PNG
- Export: SVG (clean vector geometry; **no glow**)
- Local-first: no server required

## Usage
### Run locally
1. Download `index.html`
2. Open it in a modern browser (Chrome/Edge/Firefox)

### GitHub Pages
1. Repo → **Settings** → **Pages**
2. Source: **Deploy from a branch**
3. Branch: `main` / Folder: `/ (root)`
4. Save, then open the URL GitHub provides
